This is an Alpha of Twisted. It is not an official release: there may be a number of alphas and RCs before 1.2.1 is considered a candidate for full release. Until then, use this in production at your own risk. If you want the latest stable release of Twisted, see Twisted 1.2.0, below.

Debian

Debian users can get the latest alpha, rc, or release by adding the following to their /etc/apt/sources.list:

Woody: deb http://twisted.sourceforge.net/debian/woody/ ./
Sid/Sarge: deb http://twisted.sourceforge.net/debian/ ./

FreeBSD

FreeBSD users can get Twisted releases from the official Ports collection.

NetBSD

Pkgsrc packages of Twisted are available via pkgsrc work-in-progress repository, http://pkgsrc-wip.sf.net .

Gentoo

Gentoo users can also get Twisted releases, as well as Alphas and Release Candidates, from the official distribution.

Mac OS X

Mac OS X packages (including dependencies) for the MacPython package installer can be obtained at undefined.org package repository.

Twisted Matrix Labs provides a number of Java libraries, including a Perspective Broker implementation, available on java.twistedmatrix.com.

Twisted Emacs 0.2.0

Twisted Emacs contains the beginning of an implementation of PB for elisp, as well as a development environment for hacking on Twisted with Emacs. It works on both XEmacs and GNU Emacs.
