Eye Candy

Screenshots

Every project's gotta have 'em. Twisted is particularly difficult to take a real screenshot of, since it usually lives under the hood. I've taken some screenshots of various Twisted applications in order to give you a feel for what it makes possible.

Here's my personal web server starting up, handling one request, and shutting down in a terminal. You can see a few things here.

  1. It's really easy to start up - no special installation procedure, just run twistd from wherever it's located on a pickled application object.
  2. Twisted automatically handles daemonization and logging for you; you can see it sheds root privileges immediately and automatically when it does not absolutely require them.
  3. Signal handling is automatic; your objects get startup/shutdown messages at the appropriate times.
  4. Configuration is automatically saved upon shutdown, in case it changed while you were running and you want to keep the changes.
