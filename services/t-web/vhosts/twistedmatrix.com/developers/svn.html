Subversion access to the Twisted Repository

The Twisted project is stored in a SVN repository with anonymous read-access. We have two ways to access this repository: with the web interface, and with svn.

Anonymous access to the repository

To get the SVN code anonymously, make sure you have the subversion package for your operating system, and issue the command: 

$ svn co svn://svn.twistedmatrix.com/svn/Twisted/trunk Twisted

After the command finishes, you should have a populated "Twisted" directory where you issued the 'co' command. You're all set for hacking on the bleeding-edge Twisted code-base!

Anonymous access using TortoiseSVN

Download TortoiseSVN from http://tortoisesvn.tigris.org/. Check out the svn URL svn://svn.twistedmatrix.com/svn/Twisted/trunk with it.

Remaining Informed

If you'd like to keep up to date with the changes in SVN, you can subscribe to the twisted-commits list. Archives are available.
