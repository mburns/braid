Twisted Manhole

Manhole is how you get "inside" a twisted server and crawl around. It's a set of services to allow you to poke at objects inside a running Twisted server instance and configure them.

At the lowest level, this is evaluating Python expressions inside the running server. However, manhole also provides for graphical observation of objects, and more extensions are planned for in future versions of the client.
